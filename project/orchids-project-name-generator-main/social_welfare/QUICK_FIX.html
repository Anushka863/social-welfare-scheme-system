Social Welfare System - Quick Setup

IMPORTANT: You need to start XAMPP first before accessing PHP files!

Step 1: Start XAMPP

  1. Double-click on: start_xampp.bat (in this folder)
  2. OR manually open: C:\xampp\xampp-control.exe
  3. Click Start next to Apache
  4. Click Start next to MySQL
  5. Wait for both to turn green

⚠️ Troubleshooting:
